## Features beyond install Magisk

- Remove foreground app whitelist
- Remove install apk restriction
- Enable ARM translation layer (libhoudini)

## Guide

### 1. Install Google Play Games Developer Emulator.

>[!NOTE]
> The Developer Emulator enhances the regular version by adding a convenient launcher for easy access to all apps, including Files, Google Play Store, and Chrome. You can still install games via the Play Store, just as you can with the regular version.
>
> The regular version does not have this flexible launcher, making app access and opening less convenient. To access applications like the browser, Magisk or file explorer in the regular version, you must first open Settings. Use this [guide](https://support.google.com/googleplaygamesonpc/answer/11507425) to open Settings. From there, you can navigate to other apps, for example, to download and install APK files via the browser.


### 2. Open the emulator once then close it.
### 3. Get [hpesuperpower](https://github.com/chsbuffer/hpesuperpower/releases/latest) and [Magisk](https://github.com/topjohnwu/Magisk?tab=readme-ov-file#downloads)
### 4. Run the command with Administrator Command Prompt  
```
hpesuperpower.exe magisk Magisk.apk --dev
```
